HTML Basics

<!DOCTYPE html> is used to declare a Html 5 file.
All the content that has to do with html and the layout itself belongs into the <html></html> tag.
The sections on the top of the page like <title> and the metainformation <meta charset="utf-8"> belong into the header.
Another crucial thing that is done in the header is linking to the CSS file which will be requiered later on.

            
              <head>
                <meta charset="utf-8">
                <title>Hello World!</title>
              </head>

The <body> contains the actual content of the Page. A short example of a webpage: 

            
              <!DOCTYPE html>
              <html>
                <head>
                  <meta charset="utf-8">
                  <title>Hello World!</title>
                </head>
                <body>
                  <article class="Hello_World">
                    <section>
                      <h1>Hello World</h1>
                      <p><em>Hello</em> <strong>World!</strong></p>
                      <pre style="white-space: pre-line">
                        <code>
                          Hello
                          World!
                        </code>
                      </pre>

                    </section>
                  </article>
                </body>
              </html>

CSS Basics

We already have created the basic layout.
Now its time for styling the webpage using the external Stylesheet. In our case it's called stylesheet.css.

HTML

            
              <!DOCTYPE html>
              <html>
                <head>
                  <meta charset="utf-8">
                  <title>Hello World!</title>
                  <link rel="stylesheet" href="stylesheet.css">
                </head>
                <body>
                  <article class="Hello_World">
                    <section>
                      <h1>Hello World</h1>
                      <p><em>Hello</em> <strong>World!</strong></p>
                      <pre>
                        <code>
                          Hello
                          World!
                        </code>
                      </pre>
                    </section>
                  </article>
                </body>
              </html>

CSS

          
            body {
              font-family: monospace;
            }
            h1 {
              border-left : 5px solid red;
              text-indent: 5px;
              background-color: lightgray;
            }
            article p {
              text-align: center;
              text-shadow: 2px 2px lightgrey;
            }
            article code {
                color: red;
            }

HTML Lists

Unordered List

The unoredered list is like the name already says: not oredered!

        
          <ul>
            <li>Spam</li>
            <li>and</li>
            <li>Eggs</li>
          </ul>

Ordered List

The ordered list is useful to display any sorted kind of data on the webpage.
There are different types: type="1" type="A" type="a" type="I" type="i" 

        
          <ol type="1">
            <li>Spam</li>
            <li>Eggs</li>
            <li>Sausage</li>
          </ol>

You can also create Nested list which simply means lists in lists.

CSS Lists

It's time to style your list

Using CSS you can define a margin, padding or the background color. You can also define how to display the list with display.

Here some Examples

        
          body {
            font-family: fantasy;
          }

          ul {
            list-style-type: none;
            border: 5px solid black;
            border-radius: 10px;
            background-color: gray;
            overflow: hidden;
          }

          li {
                display: block;
                float: left;
                padding-left: 10px;
          }

        
      
        
          body {
            font-family: monospace;
            background-color: gray;
          }
          ol {
            list-style-type: decimal;
            padding: 20px;
          }

          li:hover {
            font-size: 120%;
            color: white;
            background-color: #333
          }

HTML Tables

The Table

A Table is defined with the <table> tag.
You can create a row with <tr>.
You can also define a table header <th> and a cell <td> 

        
          <table>
            <tr>
              <th>Index</th>
              <th>Name</th>
              <th>Age</th>
            </tr>
            <tr>
              <td>1</td>
              <td>Newton</td>
              <td>291</td>
            </tr>
            <tr>
              <td>2</td>
              <td>Einstein</td>
              <td>138</td>
            </tr>
            <tr>
              <td>3</td>
              <td>Hawking</td>
              <td>75</td>
            </tr>
          </table>

CSS Tables

The table

Using CSS you can style your Table

        
          table {
              font-family: arial, sans-serif;
              border-collapse: collapse;
              width: 100%;
          }

          td, th {
              border: 1px solid #dddddd;
              text-align: left;
              padding: 8px;
          }

          tr:nth-child(even) {
              background-color: #dddddd;
          }

Classes and IDs

IDs

You can use IDs to format one single element via CSS.

For example:

          
            <h1 id="headline"> </h1>
          
        
          
            .headline{
              color: white;
            }

Class

You can use Classes to format all elements of it via CSS.

For example:

          
            <h1 class="black"> </h1>
          
        
          
            .black{
              color: black;
            }
